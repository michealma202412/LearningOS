# Quick Start Guide - New LearningOS App

## 🚀 Get Started in 2 Minutes

### Step 1: Open the New App

```
http://localhost:5173/app/index_new.html
```

Or if you've replaced the old file:
```
http://localhost:5173/app/index.html
```

---

### Step 2: Try Recording (30 seconds)

1. **Select mode**: Choose "🎤+📝 录音+文字" from dropdown
2. **Click**: "▶ 开始录音" button
3. **Speak**: Say something in Chinese (e.g., "今天学习的内容是...")
4. **Watch**: Text appears in real-time below
5. **Stop**: Click "⏹ 停止"
6. **Save**: Click "💾 保存"

✅ Done! You just created your first note.

---

### Step 3: View Your Files (30 seconds)

1. **Expand**: Click "📁 文件列表" header or the ▼ icon
2. **See tree**: Folders organized by date
3. **Search**: Type in the filter box to find specific items
4. **Click folder**: Expand to see individual recordings

---

### Step 4: Check Today's Review (30 seconds)

1. **Expand**: Click "🔄 今日复习" header
2. **Listen**: If you have due reviews, click ▶ to play
3. **Complete**: Click "✓ 完成" when done
4. **Collapse**: Click header again to hide

---

### Step 5: Try Different Modes (30 seconds)

#### Mode 1: Audio Only
- Select "🎤 仅录音"
- Record without text transcription
- Good for quick voice memos

#### Mode 2: Text Only  
- Select "📝 仅文字"
- Type or paste text directly
- Good for notes without audio

#### Mode 3: Audio + Text (Default)
- Both recording and text visible
- Speech converts to text in real-time
- Best for comprehensive notes

---

## 🎯 Common Tasks

### How to Create a Note with Audio + Text

```
1. Ensure mode is "🎤+📝 录音+文字"
2. Click "▶ 开始录音"
3. Speak your content
4. Watch text appear automatically
5. Edit text if needed (fix any recognition errors)
6. Click "⏹ 停止"
7. Click "💾 保存"
```

### How to Find Old Recordings

```
1. Expand "📁 文件列表" section
2. Use search box to filter by keyword
3. Click folder date to expand
4. See all items in that folder
```

### How to Complete a Review

```
1. Expand "🔄 今日复习" section
2. Find today's due items
3. Click ▶ to listen
4. Click "✓ 完成" when finished
5. Item disappears from list
```

### How to Generate QR Code

```
1. Click "📱 二维码" button in header
2. Copy the URL shown
3. Use online QR generator (e.g., qr-server.com)
4. Print or save the QR code
```

## ❓ Quick Troubleshooting

### Can't hear playback?
→ Check system volume and browser permissions

### Speech recognition not working?
→ Use Chrome browser, check internet connection

### Don't see old recordings?
→ Refresh page (Ctrl+R), check IndexedDB has data

### Text not appearing while speaking?
→ Ensure mode is "录音+文字", microphone allowed

### Can't save?
→ Check both audio and text aren't empty

## 🎓 Video Tutorial (Text Version)

### Scenario 1: Recording a Lecture

```
1. Open app before class starts
2. Select "🎤+📝 录音+文字" mode
3. When lecture begins, click "▶ 开始录音"

```

```
2. Expand "🔄 今日复习" section

   c. Click "✓ 完成"

```
