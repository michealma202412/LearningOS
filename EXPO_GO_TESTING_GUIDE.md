# LearningOS Expo Go 快速原型验证指南

## 🎯 方案说明

Expo Go 是一个无需 macOS 即可在 iOS/Android 上快速测试 React Native 应用的方案。

**优势**:
- ✅ 无需 macOS 电脑
- ✅ 无需 Xcode
- ✅ 扫码即可运行
- ✅ 快速原型验证

**限制**:
- ⚠️ 部分原生功能受限（如稳定录音）
- ⚠️ 数据仅保存在内存中（重启丢失）
- ⚠️ 性能不如原生应用

## 🚀 使用步骤

### 第1步：等待服务器启动

Expo 服务器启动后会显示二维码和 URL：

```
┌──────────────────────────────────────────────┐
│                                              │
│  Your QR code is ready!                      │
│                                              │
│  To run your app:                            │
│  • Scan the QR code with Expo Go             │
│  • Or use connection URL below               │
│                                              │
│  exp://xxx.xxx.xxx.xxx:19000                 │
│                                              │
└──────────────────────────────────────────────┘
```

### 第2步：在手机上安装 Expo Go

#### iOS (iPhone):
1. 打开 App Store
2. 搜索 "Expo Go"
3. 点击"获取"安装
4. 图标为紫色背景白色"E"

#### Android:
1. 打开 Google Play Store
2. 搜索 "Expo Go"
3. 点击"安装"

### 第3步：扫码运行应用

#### 方法 A: 扫描二维码（推荐）
1. 打开 Expo Go App
2. 点击 "Scan QR Code"
3. 扫描终端显示的二维码
4. 应用会自动下载并运行

#### 方法 B: 手动输入 URL
1. 打开 Expo Go App
2. 点击 "Enter URL manually"
3. 输入终端显示的 URL（如 `exp://192.168.x.x:19000`）
4. 点击 "Load"

### 第4步：测试功能

应用启动后，你可以测试：

#### ✅ 可用功能
- [ ] **底部导航切换** - 记录 / 文件 Tab
- [ ] **标题输入** - 输入记录标题
- [ ] **模拟录音** - 点击"开始录音"按钮
- [ ] **文本编辑** - 编辑识别结果
- [ ] **保存记录** - 保存到内存
- [ ] **查看列表** - 切换到文件 Tab 查看
- [ ] **删除记录** - 点击垃圾桶图标删除

#### ⚠️ 模拟功能说明
- **录音**: 点击后会模拟 3 秒录音，然后自动生成示例文本
- **数据存储**: 所有数据保存在内存中，关闭应用后丢失
- **SQLite**: 未启用（Expo Go 限制）
- **文件系统**: 未启用（Expo Go 限制）

## 🐛 常见问题

### Q1: 二维码无法扫描
**症状**: Expo Go 提示无效二维码

**解决**:
1. 确保手机和电脑在同一 WiFi 网络
2. 尝试使用方法 B（手动输入 URL）
3. 检查防火墙是否阻止了连接

### Q2: 连接超时
**症状**: 加载时显示 "Connecting to Metro..."

**解决**:
1. 检查手机和电脑是否在同一网络
2. 关闭手机 VPN
3. 重启 Expo 服务器: `Ctrl+C` 然后重新运行 `npx expo start`

### Q3: 应用闪退
**症状**: 打开后立即关闭

**解决**:
1. 更新 Expo Go 到最新版本
2. 清除 Expo Go 缓存
3. 重新扫描二维码

### Q4: 看不到最新更改
**症状**: 修改代码后手机没反应

**解决**:
1. 在 Expo Go 中摇动手机
2. 选择 "Reload"
3. 或按 `R` 键在终端刷新

### Q5: 录音功能不工作
**说明**: 这是预期行为！

Expo Go 原型中的录音是**模拟演示**，因为：
- Expo Go 沙盒环境限制了原生录音插件
- 需要完整 Capacitor 方案才能实现稳定录音

在终端可以看到应用日志：


### 强制刷新
- **iOS**: 摇动手机 → Reload

### 开发者菜单
摇动手机可打开开发者菜单：
- Reload - 重新加载
