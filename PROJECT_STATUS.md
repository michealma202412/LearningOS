## 🎯 使用场景

### 场景1: 个人学习
```
学生在学习本上贴着每天的二维码
每天扫一个二维码 → 进入当天的录音页 → 记录学习疑惑
系统自动按艾宾浩斯曲线提醒复习
```

### 场景2: 家庭教育
```
家长为孩子生成周一到周五的学习链接
周日发送 5 个二维码给孩子
孩子每天扫一个 → 自动打开预定的学习任务
家长可以查看 folder.html 了解学习进度
```

### 场景3: 学校教学
```
老师生成班级整周的学习二维码墙
贴在教室复习区
学生课后扫不同日期的二维码进行分散复习
充分利用艾宾浩斯遗忘曲线
```

### 场景4: 鼓励和奖励
```
孩子按时完成复习任务 → 标记"✓ 完成"
UI 显示绿色进度条和完成统计
家长可以设置奖励机制
```
