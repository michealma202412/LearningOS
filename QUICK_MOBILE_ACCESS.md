## ⚠️ 常见问题

### Q1: 手机无法访问？

**检查清单：**
- [ ] 手机和电脑是否在同一WiFi？
- [ ] 电脑防火墙是否阻止连接？
- [ ] Vite服务器是否在运行？

**解决方法：**
```bash
# 重新查看IP地址
ipconfig

# 重启开发服务器
npm run dev -- --host
```

### Q2: 连接超时？

**尝试：**
1. 关闭Windows防火墙测试
2. 使用其他IP地址（上面列出的三个）
3. 检查路由器设置

### Q3: 页面加载慢？

**优化：**
- 靠近路由器
- 切换到5GHz WiFi
- 减少同时连接的设备

## 💡 重要提示

### 1. 保持电脑运行
- ❌ 关闭终端 = 手机无法访问
- ❌ 电脑睡眠 = 手机无法访问
- ✅ 必须保持Vite服务器运行

### 2. IP地址可能变化
- 重启路由器后IP会变
- 重新运行 `npm run dev -- --host` 查看新IP

### 3. 数据隔离
- 每个设备独立存储数据
- 手机和电脑的数据不共享
- 可以使用不同用户登录
