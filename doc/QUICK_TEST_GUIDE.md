## 🐛 常见问题排查

### 问题 1：folder.html 仍然显示"加载中..."

**排查步骤：**
1. 打开 F12 控制台
2. 查看是否有红色错误信息
3. 检查 IndexedDB 是否正常

**可能原因：**
- IndexedDB 未正确初始化
- 数据库版本冲突
- JavaScript 语法错误

**解决方法：**
```javascript
// 在控制台执行：
indexedDB.deleteDatabase('learning_os')
// 然后刷新页面
```

---

### 问题 2：语音识别不工作

**检查清单：**
- [ ] 使用 Chrome 浏览器（不是 Safari/Firefox）
- [ ] 使用 localhost 或 HTTPS（不是 file://）
- [ ] 允许了麦克风权限
- [ ] 网络连接正常

**替代方案：**
- 手动输入文章内容
- 使用外部语音转文字工具

---

### 问题 3：导出时只下载了 Markdown，没有音频

**原因：**
浏览器可能会阻止多个文件同时下载

**解决方法：**
1. 允许浏览器的多个下载
2. 或者逐个确认每个下载
3. 代码已添加 500ms 延迟避免此问题

---

### 问题 4：文章查看页显示空白

**检查：**
1. URL 中是否有 `?id=xxx` 参数
2. 控制台是否有错误
3. 数据库中是否存在该文章

**调试：**
```javascript
// 在 article.html 控制台执行：
const urlParams = new URLSearchParams(window.location.search);
console.log('Article ID:', urlParams.get('id'));
```
